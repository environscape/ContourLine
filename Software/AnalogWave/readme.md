# AnalogWave
base on Arduino nano 328p  
for ContourLine 2.1  
作者: lecheng  
说明: 该振荡器是模仿模拟振荡器来设计的   
演示: https://www.bilibili.com/video/BV17y411B729  

### 音频接口:  

0.      a0:VOCT                   a1:FM Amount  
1.      a2:Detune Amount          a3:WaveType
2.      d11:Hard Sync             d9d10:AudioOut    

### 显示参数:  

 0.         8位数码管显示字母        参数名词EN                   参数名词CN                    参数说明
 1.         P                       Pitch                       音高                          C1-A2
 2.         R                       Rang                        八度
 3.         F                       FM Amount                   频率调制量
 4.         S                       Sub                         倍低频量                    
 5.         D                       Detune                      音飘程度                  
 6.         T                       WaveType                    波形选择
