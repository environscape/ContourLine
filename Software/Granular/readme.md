# 粒子合成器
# Granular
### base on Arduino nano 328p/168p
#### for ContourLine 2.1  
作者: lecheng  
演示：https://www.bilibili.com/video/BV1ew4m117T7


### 音频接口:  

0.      a0:Freq                         a1:Density
1.      a2:Seed Lv                      a3:Seed Density  
2.      d11:Null                        d9d10:AudioOut    

### 显示参数:  



 0.         8位数码管显示字母        参数名词EN                   参数名词CN                    参数说明
 1.         1                       Freq                        频率                      
 2.         2                       Density                     密度
 3.         3                       Diffusion                   弥散 密度随机量
 4.         4                       feedback                    重复密度 
 5.         5                       level                       音量的随机程度
